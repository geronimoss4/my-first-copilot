
##🧩 Modos do Copiloto (Ask, Edit, Plan, Agent e Study)

![dio/me](https://img.shields.io/badge/dio-me-ff2d55)
![IA](https://img.shields.io/badge/IA-Assistente%20Inteligente-blue)
![Prompt](https://img.shields.io/badge/Prompt-engineering-yellow)

O Copiloto oferece diferentes **modos de interação** para se adaptar ao seu fluxo de trabalho — desde **tirar dúvidas**, até **editar código**, **planejar soluções** ou **automatizar tarefas complexas**.

A proposta é simples: escolher o modo certo para cada momento e ganhar mais **velocidade, controle e eficiência**.



## ❓ Ask

Use o modo **Ask** para **entender o código sem modificá-lo**.

Você pode perguntar sobre:

* funções específicas
* erros e stack traces
* arquivos do projeto
* conceitos técnicos

O Copiloto atua como um **mentor técnico**, analisando o contexto e explicando o que está acontecendo — sem alterar nada.

📄 **Prompt:** `prompts/prompt-ask.md`



## ✏️ Edit

O modo **Edit** é usado para **modificar código existente**.

Você seleciona um trecho (ou arquivo) e descreve a mudança desejada.

Ideal para:

* refatoração
* ajustes de lógica
* melhorias de performance
* padronização de código
* tratamento de erros

Foco: **transformar o que já existe**.

📄 **Prompt:** `prompts/prompt-edit.md`



## 🧭 Plan

Para tarefas mais complexas, use o modo **Plan**.

Ele:

* divide o problema em etapas
* explica a abordagem
* valida a estratégia antes da execução

Perfeito para:

* novas funcionalidades
* mudanças estruturais
* decisões arquiteturais

📄 **Prompt:** `prompts/prompt-plan.md`



## 🤖 Agent

O modo **Agent** é mais autônomo e orientado a execução.

Ele pode:

* navegar pelo projeto
* criar e editar múltiplos arquivos
* manter contexto entre ações

Você define o objetivo, e ele executa como um **dev assistente**.

📄 **Prompt:** `prompts/prompt-agent.md`



## 📚 Study

O modo **Study** é focado em **aprendizado profundo**.

Ele:

* ensina o raciocínio por trás das soluções
* destaca conceitos importantes
* explora trade-offs
* conduz passo a passo

Funciona como um **tutor técnico interativo**.

📄 **Prompt:** `prompts/prompt-study.md`



## 🧠 Resumo rápido

* **Ask** → entender
* **Edit** → modificar
* **Plan** → estruturar
* **Agent** → executar
* **Study** → aprender
